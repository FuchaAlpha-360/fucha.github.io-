# Niwuh Alpha - Portfolio Website

A professional portfolio website showcasing full-stack development skills and projects.

## Features

- **Responsive Design**: Mobile-friendly layout using Bootstrap 5
- **Multi-page Navigation**: Home, About, and Contact pages
- **Contact Form**: Integrated with Formspree for form submissions
- **Modern UI**: Clean and professional design with smooth interactions

## Deployment Instructions

### Option 1: GitHub Pages

1. Push your repository to GitHub
2. Go to repository **Settings** → **Pages**
3. Select the branch to deploy (typically `main`)
4. Your site will be available at `https://yourusername.github.io/repository-name`

### Option 2: Netlify

1. Push your repository to GitHub
2. Visit [netlify.com](https://netlify.com) and sign in
3. Click **New site from Git** and connect your GitHub repository
4. Netlify will automatically build and deploy your site

### Option 3: Vercel

1. Push your repository to GitHub
2. Visit [vercel.com](https://vercel.com) and sign in
3. Click **New Project** and import your GitHub repository
4. Vercel will automatically deploy your site

### Option 4: Self-Hosted Server

1. Upload all files to your web hosting provider via FTP/SFTP
2. Ensure the web server serves `index.html` as the default document
3. Test all pages and links
